## 特性

- 开源：MIT 协议，代码透明可审计，接受社区贡献
- 免费：无广告、无内购，所有功能永久免费无限制
- 美观：Material You 动态主题，支持深色模式与多套配色
- 强大：集成解压、编辑、播放、APK，以及Shizuku 支持

## 开发规范

本项目遵循 [《ZhuFiler 五章十九条开发规范》](./ZhuFiler_Spec.md)，所有代码提交须合规。
